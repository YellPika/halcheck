Detailed Description

template<typename T>
struct halcheck::gen::is_arbitrary< T >

Determines if a type supports generating arbitrary values.

Template Parameters
TThe type of value to query.

The type T satisfies is_arbitrary if and only if it is not a reference type and the expression arbitrary(gen::tag<T>(), lib::atom()) returns a value convertible to type T. Note that, by default, this holds for the following cases:

  • T is lib::monostate
  • T is bool
  • std::is_integral<T>() holds.
  • std::is_floating_point<T>() holds.
  • T is std::pair<U, V>, where gen::is_arbitrary<U>() and gen::is_arbitrary<V>() hold
  • T is std::tuple<Args...>, where gen::is_arbitrary<U>() holds for all U in Args...
  • T is lib::optional<U>, where gen::is_arbitrary<U>() holds
  • T is std::variant<Args...>, where gen::is_arbitrary<U>() holds for all U in Args...
  • lib::is_insertable_range<T>() and gen::is_arbitrary<lib::range_value_t<Container>>() hold
