A random element of range.

A call to this function is well-formed only if one of the following is true:

  • lib::is_input_range<T>() and lib::is_sized_range<T>() hold.
  • lib::is_forward_range<T>() holds.

Overload (1) is invoked if T is an l-value reference and lib::is_forward_range<T> holds. Otherwise, overload (2) is invoked.

Note
Overload (1) is called if T is an l-value reference and lib::is_forward_range<T> does not hold because the value obtained from an input iterator is not guaranteed to exist once the iterator is modified or destroyed.
